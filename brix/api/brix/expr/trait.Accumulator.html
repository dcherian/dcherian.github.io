Expand description

Fold accumulator for reductions.

State is type-erased via Box<dyn State> / &mut dyn State. Each accumulator defines its own concrete state struct with typed fields and downcasts internally.

fn init(&self, coord: &BrickCoordinate) -> BrixResult<Box<dyn State>>

Create initial state for the given output chunk coord. Concrete impls derive bounds internally from their baked-in ChunkGrid — fallible because deriving the brick shape requires the coord to carry every axis the grid expects.
