Brick
A located chunk of data: data plus its block position in the producer’s chunk grid.
BrickCoordinate
A brick’s position in the producer’s chunk grid — a per-axis block index keyed by axis name, with insertion order preserved. Order matches the payload’s shape, so iter yields axes in positional order; get looks up by name. Carrying the names makes bricks self-describing: kernels can read each input’s dim order straight off brick.coord and align/broadcast accordingly, without the kernel having to memorize producer dims.
