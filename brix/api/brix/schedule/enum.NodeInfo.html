Source

Unified source node. Concrete Source impls (in array/) bake their element-sized ChunkGrid in and stamp each brick’s bounds at production time; brix sees only the counts-only IterationSpace.

Blockwise

Blockwise computation. Per-input chunk-coord routing is carried on each edge’s ChunkAccess (set by lowering) and propagated up via the planner’s coord transforms onto each producer’s Apply.access.
