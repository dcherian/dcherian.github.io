Sketch
A compact, PartialEq-able representation of a schedule tree. Used for pretty test assertions. Shows each node’s NodeId — which is the user-supplied label when Array::labeled(...) was used, otherwise the op-kind prefix ("source", "blockwise", "reduce", "sink", …).
