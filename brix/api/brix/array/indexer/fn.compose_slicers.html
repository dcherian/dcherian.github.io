pub fn compose_slicers(

Expand description

Compose two per-axis indexer maps applied as inner then outer: equivalent to expr.isel(inner).isel(outer). Outer slicing operates within the inner window.

Per axis (Slicer-only, step = 1):

  inner = Slicer { start: s_i, end: e_i, step: 1 }
  outer = Slicer { start: s_o, end: e_o, step: 1 }
  composed = Slicer { start: s_i + s_o, end: s_i + e_o, step: 1 }

Axes present in only one of the two maps are kept verbatim (the other side is identity on that axis). Future: step ≠ 1 composes as composed.start = s_i + s_o * t_i, composed.step = t_i * t_o, composed.end = s_i + e_o * t_i (lands when the planner accepts non-unit step).
