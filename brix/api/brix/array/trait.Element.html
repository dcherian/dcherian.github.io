Expand description

Native primitive backing a TypedArray / Value arm. The static mirror of the runtime DType tag: T::DTYPE is the tag, from_array/from_scalar re-tag a concrete buffer/scalar, and from_value peels the tag back off.

to_le_bytes/from_le_bytes use a fixed 8-byte little-endian frame for all primitives (both current dtypes are 8 bytes); the dtype-tagged header in brick.rs records which T the bytes belong to.
