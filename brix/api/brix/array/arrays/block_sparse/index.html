BlockSparseArray — array-layer wrapper for arrays with storage sparsity (some chunks present, others absent).

Sibling type to Array. Shares the same underlying ArrayExpr / mid-IR machinery, but additionally carries a fill_value and gates ops that aren’t defined yet at op-construction time. V0 supports: construction from an OpaqueSource whose native_grid carries occupancy, unary map, and reduce-along-dense-axis. Sparse-axis reduce, slicing along a sparse axis, and any binary op are hard-errored.
