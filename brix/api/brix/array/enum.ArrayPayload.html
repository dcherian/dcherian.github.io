pub fn get_frame_mut<T: Element>( &mut self, axes_idx: &HashMap<usize, usize>, ) -> WriteFrame<'_, T>

Grab a mutable frame at axes_idx, typed as T.

axes_idx maps each framed axis (by positional index into this payload) to the frame index along that axis. Non-framed axes (loop dims) are kept at full extent.

Promotes Fill → Dense on the way in. Panics if T does not match the payload’s dtype — callers (accumulators) dispatch on the dtype first, so the tag always matches.

pub fn axis_chunks_iter( &self, axis: usize, size: usize, ) -> impl Iterator<Item = ReadFrame<'_>> + '_

Iterate sub-views by splitting axis into contiguous chunks of size size. Matches ndarray’s axis_chunks_iter semantics — the last chunk may be smaller.

Source

pub fn as_frame_iter<'a>( &'a self, view: HashMap<usize, Frame>, ) -> impl Iterator<Item = (HashMap<usize, usize>, ReadFrame<'a>)> + 'a

Iterate (axes_idx, slot) pairs cut out of this brick.

view carries one entry per reduce axis, keyed on the brick-position of that axis. The iterator yields the cartesian product of per-axis frames in lexicographic order of axis position.

pub fn align( inputs: &[(&ArrayPayload, &BrickCoordinate)], target: &IndexMap<AxisName, (usize, BroadcastMode)>, ) -> Vec<ArrayPayload>

Align a list of payloads to a common target shape.

target maps each target dim name to (extent, broadcast_mode) in positional order. Each input pairs a payload with its positional axis names via a BrickCoordinate. Returns aligned payloads in the same order as inputs. For each input axis: axes present in target are permuted into place; axes missing from the input get inserted as size-1 broadcast axes. Extent checks follow each target dim’s BroadcastMode. Fill stays a Fill at the target shape (no allocation beyond the shape vec); Dense permutes / inserts axes / broadcasts and materializes to owned storage.

One-input convenience for ArrayPayload::align: align self (paired with input_coord) to target.
