fn construct_sliced_output_chunk_grid(

Expand description

Per-axis: input AxisChunks + slice [start, end) step step → (output AxisChunks, offset: source-chunk index of output chunk 0).

Each output chunk corresponds to one source chunk (sometimes partial, always RLE-encoded). The interesting variant transitions are Uniform → RLE (slice introduces boundaries) and RLE → RLE (slice composes onto an already-segmented axis).

Caller is responsible for validating that the slice is in-bounds against the axis extent (Slice::output_extent); this function assumes end <= input.extent().
