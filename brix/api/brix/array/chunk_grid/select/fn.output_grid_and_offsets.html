pub fn output_grid_and_offsets(
    input_grid: &ChunkGrid,

Expand description

Compute the output ChunkGrid for a Slice op given input grid and indexers. Returns (output_grid, chunk_offsets) where chunk_offsets[d] is the source-chunk index that output chunk 0 maps to along axis d.

Axes not in slicers are passed through unchanged. For sliced axes, each output chunk maps to one source chunk after the offset.
