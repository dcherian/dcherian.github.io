Chunk-grid algebra for orthogonal indexing (BlockwiseOp::Gather).

Plan-time math: given an input ChunkGrid and per-axis 1-D integer index arrays, partition each axis into runs by source chunk and build the output ChunkGrid. The brick-level kernel that gathers one output brick lives in crate::array::GatherKernel.

Output chunking is per-source-chunk-touched: each output chunk corresponds to one source chunk along the indexed axis; the chunk holds the in-chunk positions hitting that source chunk. This is the sparse, vindex-style runs model — only the source chunks the indices actually touch are read at execute time.
