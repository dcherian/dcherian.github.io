Built-in accumulators for ArrayOp::Reduce.

ArrayAccumulator carries the user-facing metadata (dtype + reduction axis names). Lowering wraps the user-facing accumulator in a LoweredAccumulator that includes the output ChunkGrid and implements brix::Accumulator.

Computation over a single brick is serial. We leave parallelism to across-brick.

Things I don’t like

  • we have a generic State. This requires a state.as_any_mut().downcast_mut::<SumState>().unwrap() dance for each accumulator here.
