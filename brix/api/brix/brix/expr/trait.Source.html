Expand description

A source of brick payloads. Concrete impls (in array/) bake their element-sized ChunkGrid in as a private field; get_brick_payload returns just the payload for the given chunk coord, and the executor re-attaches coord to build a Brick.

brix never names ChunkGrid directly — the planner reads only chunk counts from the surrounding Expr::space.
