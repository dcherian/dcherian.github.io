Mid-level IR — chunk-grid-aware operator graph and physical schedule.

This is the pipeline-agnostic layer. Lives below crate::array (the logical, dtype-aware layer) and exposes a small structural vocabulary: Source / Kernel / Accumulator traits, an Expr operator graph, a Schedule loop nest, and the planner that lowers between them.

Rule: modules in brix/ must not import from crate::array. Dependency is one-way: array → brix.
