1//! Array data layer — logical arrays, their mid-IR lowering, and the concrete
2//! brick payloads / kernels / accumulators they lower to.
3//!
4//! The user-facing array kinds (`Array`, `BlockSparseArray`) and their shared
5//! `ArrayLike` contract live in `arrays`; this module root wires the
6//! submodules together and re-exports the public surface. The per-dtype trait
7//! instantiations live in `generated/` (a folder convention, not codegen).
8
