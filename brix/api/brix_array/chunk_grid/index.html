broadcast_grids
Derive an output ChunkGrid by aligning input grids to output_dims by name. For each output dim, find the first input that carries that named axis with size > 1 and inherit its chunks; the extent is the max across inputs that carry the name (broadcast rule: 1 yields to non-1). If no input has the name at all, the axis is a single-chunk size-1 broadcast.
