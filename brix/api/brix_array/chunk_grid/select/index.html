brix_array::chunk_grid

Module select 

Source
Expand description

Chunk-grid algebra for positional selection (BlockwiseOp::Subset).

Plan-time math: given an input ChunkGrid and a per-axis indexer map, produce the output ChunkGrid plus the per-axis source-chunk offsets that propagate as coord transforms to producers. The brick-level kernel that clips a producer brick lives in [brix::SliceKernel].
