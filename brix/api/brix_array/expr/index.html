ArrayExpr — the array-level expression layer.

Array expressions include the “usual” primitive array ops

  • Indexing: select (basic indexing), oindex (orthogonal indexing)
  • reduction
  • blockwise for map

and a few convenient specializations

The optimizer rewrites ArrayExpr to ArrayExpr. The optimized expression is lowered to “mid-IR” brix::Expr. At the mid-IR layer, only num-chunks is needed. The planner (brix/planner.rs) converts mid-IR to a Schedule.
