Lower the array-level IR (ArrayExpr) to the mid-IR (Expr).

Walks ArrayExpr post-order and emits the corresponding Expr.

To do so, we need to also make concrete ChunkGrid choices for execution. That information is embedded in the brix::Source, brix::Kernel, brix::Accumulator passed on to brix.

Note that the lowering here is a mechanical translation from ArrayExpr to Expr. We simply add concrete ChunkGrid information to the outputs of each Expr The planner in brix/planner.rs is responsible for wiring this up and accounting for broadcasting semantics etc.

We have expect in this file, because we expect those invariants to be enforced higher up the call chain.
